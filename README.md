# Portfólio de Projetos Full Stack e BI

Este repositório reúne 3 projetos prontos para portfólio:

1. **Sistema de Gestão de Pedidos** — Java, Spring Boot, React, PostgreSQL
2. **Plataforma de Agendamentos** — Java, Spring, MySQL, Angular, WebSocket, Docker, AWS
3. **Dashboard de Faturamento** — Power BI e SQL

## Estrutura

```bash
.
├── sistema-gestao-pedidos
├── plataforma-agendamentos
└── dashboard-faturamento
```

## Objetivo

Montar um portfólio com projetos que mostram backend, frontend, autenticação, tempo real, CI/CD, banco de dados e análise de dados.
