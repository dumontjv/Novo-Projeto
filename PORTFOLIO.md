# Portfólio de Projetos

Este repositório reúne três projetos para apresentação profissional no GitHub.

## Projetos

### 1. Sistema de Gestão de Pedidos
Stack: Java, Spring Boot, React, PostgreSQL

Entrega principal:
- API REST
- autenticação e segurança inicial
- dashboard operacional
- banco com Docker Compose

Link:
- `sistema-gestao-pedidos/`

### 2. Plataforma de Agendamentos
Stack: Java, Spring Boot, MySQL, Angular, WebSocket

Entrega principal:
- gestão de agendamentos
- atualização em tempo real
- backend com MySQL
- frontend Angular base

Link:
- `plataforma-agendamentos/`

### 3. Dashboard de Faturamento
Stack: Power BI, SQL

Entrega principal:
- base analítica
- consultas SQL
- medidas DAX
- wireframe e storytelling executivo

Link:
- `dashboard-faturamento/`

## Objetivo de portfólio
Mostrar domínio em backend, frontend, bancos relacionais, integração em tempo real e análise de dados.
