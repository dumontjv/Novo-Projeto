# Agência LeadForge

Projeto front-end premium de uma agência de marketing com foco em **conversão de leads**, **autoridade digital** e **performance comercial**.

## Visão do projeto

A proposta da LeadForge é responder a uma pergunta que todo cliente faz em silêncio antes de contratar uma agência:

> **"Isso vai me trazer cliente?"**

Por isso, este projeto foi desenhado como uma landing page institucional moderna, rápida e responsiva, com estrutura orientada a conversão, prova social, blocos de serviços, métricas, FAQ e formulário de captação.

## Tecnologias

- React 18
- Vite
- CSS puro com arquitetura modular por seções
- Metadados básicos de SEO

## Destaques para currículo

- Desenvolvimento de interface responsiva com foco em conversão
- Arquitetura componentizada em React
- Organização de conteúdo orientada a marketing e UX
- Estrutura pronta para evolução em integrações reais
- Implementação de landing page comercial com copy estratégica

## Funcionalidades

- Hero section com proposta de valor clara
- Blocos de serviços para tráfego, landing pages e automação
- Área de métricas e resultados
- Prova social e FAQ
- Formulário de contato pronto para integração futura
- Layout totalmente responsivo
