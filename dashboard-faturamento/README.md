# Dashboard de Faturamento

## Stack
- Power BI
- SQL

## Objetivo
Criar um painel com visão executiva de faturamento, margem, crescimento mensal, ticket médio e participação por categoria.

## Conteúdo
- Script SQL para criação da base
- CSV de exemplo com dados financeiros
- Guia de modelagem para Power BI

## KPIs sugeridos
- Receita total
- Receita por mês
- Margem bruta
- Ticket médio
- Top 5 clientes
- Top 5 categorias
- Comparativo mês contra mês
