# Layout sugerido do Dashboard de Faturamento

## Página 1 - Visão Executiva
- Card de Receita Total
- Card de Custo Total
- Card de Margem Bruta
- Card de Margem Percentual
- Linha de evolução mensal da receita
- Barras com Top 5 clientes
- Rosca com participação por canal

## Página 2 - Performance Comercial
- Barras por categoria
- Matriz por cliente e categoria
- Mapa por estado
- Segmentadores por mês, canal e estado

## Página 3 - Rentabilidade
- Tabela com receita, custo e margem por produto
- Scatter plot para receita x margem
- Análise comparativa mês contra mês

## Paleta sugerida
- Fundo claro
- Cards neutros
- Destaque para métricas positivas
- Uso moderado de contraste para leitura executiva
