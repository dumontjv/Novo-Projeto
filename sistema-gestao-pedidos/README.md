# Sistema de Gestão de Pedidos

## Stack
- Java 21
- Spring Boot 3
- Spring Security + JWT
- PostgreSQL
- React + Vite

## Funcionalidades
- Login com JWT
- Controle de acesso por perfil (`ADMIN`, `MANAGER`, `USER`)
- CRUD de pedidos
- Atualização de status do pedido
- Dashboard com pedidos por status, prioridade e cliente
- Base preparada para atualização em tempo real
