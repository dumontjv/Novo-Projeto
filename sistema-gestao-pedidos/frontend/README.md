# Frontend - Sistema de Gestão de Pedidos

## Requisitos
- Node.js 20+
- npm 10+

## Como rodar
1. Entre na pasta frontend
2. Execute npm install
3. Execute npm run dev

## Fluxo esperado
- Acesse http://localhost:5173
- Faça login com as credenciais demo
- Visualize o dashboard e os pedidos carregados do backend
